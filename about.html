Pokemon Game

Me and one of my friends created a Pokemon game with C Language which is played between a user and the computer. The computer selects a pokemon at random. The user gets to choose his pokemon for the battle. The computer does any one of the four attacks at random with the help of random number generator. Both of the pokemons (user's and computer's) has a total of 100 HP. after every attack, the HP decreases by a random number. Whoever manages to reduce HP of the opposition pokemon to 0, wins the battle.
The source code is available on my github repository.
